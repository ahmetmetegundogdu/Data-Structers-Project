## Artificial Intelligence Fields – Data Structures Project

This project was developed for a **Data Structures** course assignment.

It models the **main fields of Artificial Intelligence** using a **Binary Search Tree (BST)**.  
Each main field node contains another BST that stores its **subfields**, ordered alphabetically.

### Implemented Features
- AI main fields and subfields generated using a **Generative AI prompt**
- Each subfield includes a **definition** and **application areas**
- Alphabetical listing of main fields and their subfields
- Detection of the **deepest subfield tree** and its depth
- Word frequency analysis on subfield definitions using a separate BST

### Concepts Used
- Binary Search Trees
- Tree traversal and depth calculation
- String parsing and word counting

### Purpose
To apply tree-based data structures to a real-world AI domain example.
